# CPU命名规则

目前市面上主流的**CPU**厂商为：**Intel**和**AMD**。
## Intel

以`intel Core i7-13700K`为例，其中：
`intel`: 品牌名
`Core`: 产品线名
`i7`: 型号名
`13`: 代数
`700`: 子型号
`K`: 尾缀

## AMD

以`AMD Ryzen7 7700 X`为例，其中：
`AMD`: 品牌名
`Ryzen7`: 产品线及定位
`7`: 代数
`700`: 子型号
`X`: 尾缀
