# 哈希
python的字典和集合是基于哈希实现的，哈希具有查询时间复杂度O(1)的性质
哈希指的是当插入元素时，通过哈希函数计算元素的哈希值，并根据哈希值决定元素插入的位置，如果该位置已经有元素了，则使用链表放在该元素后面
