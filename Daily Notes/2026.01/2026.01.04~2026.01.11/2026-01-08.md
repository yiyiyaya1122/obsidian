## 跑通`quest3s`库

```bash

# 更新apt包列表，确保获取最新的软件包信息
sudo apt update

# 安装pixi
curl -fsSL https://pixi.sh/install.sh | bash

# 验证Pixi安装成功
pixi --version

# 删除项目目录中的.pixi缓存目录，避免旧环境干扰
rm -rf ~/.pixi

# 根据pixi.toml和pixi.lock文件安装所有依赖
# --verbose 显示详细安装信息
pixi install --verbose
# 如果遇到网络问题，可以尝试：
pixi install --no-cache  # 不使用缓存

# 使用Pixi运行colcon构建命令
# --symlink-install 创建符号链接而非拷贝，便于开发调试
# --packages-select [包名] 可选：仅编译指定包
pixi run colcon build --symlink-install --cmake-args -DCMAKE_BUILD_TYPE=Release

# 仅编译特定包
pixi run colcon build --symlink-install --packages-select your_package_name

# 排除某些包
pixi run colcon build --symlink-install --packages-ignore package_to_ignore

# 验证编译结果
source install/setup.bash
pixi run ros2 pkg list  # 查看已编译的包

```
